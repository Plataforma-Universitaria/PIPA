PIPA

Plataforma Integrada Personalizável para Assistentes

É uma plataforma digital inovadora que integra sistemas acadêmicos a assistentes virtuais, possibilitando que as instituições parceiras criem outros canais de acesso aos serviços oferecidos pelas instituições de ensino através de chatbots e assistentes virtuais.

Sobre o Projeto

Objetivo Principal

O projeto PIPA propõe o desenvolvimento de uma plataforma digital voltada para instituições de ensino, com o objetivo de integrar sistemas acadêmicos tradicionais a assistentes virtuais. A plataforma busca facilitar o acesso dos usuários aos serviços oferecidos pelas instituições parceiras, por meio da utilização ativa de modelos de linguagem (LLMs) no processamento de intenções e na comunicação das informações.

Estudo de Caso

O estudo de caso foi realizado na, Universidade Estadual de Goiás, abrangendo desde a fundamentação teórica até a modelagem e o planejamento da implantação da plataforma. Esta abordagem representa uma aplicação estratégica dessas tecnologias no contexto educacional.
Dentre os serviços oferecidos pelo nosso estudo de caso, estão:

  • Consulta de aulas
  • Consulta de notas
  • Consulta de faltas

Inovação Tecnológica

A PIPA representa uma aplicação estratégica de tecnologias de ponta no contexto educacional, combinando assistentes virtuais com sistemas acadêmicos para criar uma experiência de usuário mais intuitiva e eficiente. Pipa traz uma proposta de ferramentalização de LLMs, explorando a sua capacidade lógica além da conversacional.

Principais Funcionalidades

Assistentes Virtuais Inteligentes

Integração de assistentes virtuais avançados capazes de processar linguagem natural e fornecer respostas precisas aos usuários.

Processamento com LLMs

Utilização de Large Language Models para processamento de intenções e comunicação eficiente das informações.

Integração de Sistemas

Conexão seamless entre sistemas acadêmicos tradicionais e interfaces modernas de assistentes virtuais.

Foco Educacional

Plataforma especialmente desenvolvida para atender às necessidades específicas de instituições de ensino.

Personalizável

Arquitetura flexível que permite personalização conforme as necessidades de cada instituição parceira.

Experiência do Usuário

Interface intuitiva que facilita o acesso dos usuários aos serviços oferecidos pelas instituições.

Tecnologia e Arquitetura

Componentes Principais

  • API_IA: Interface para integração com modelos de linguagem
  • MIDDLEWARE: Componente responsável pelo gerênciamento de tokens
  • INTEGRATOR: Componente de integração entre o core e os provedores existentes
  • PIPA_CORE: Componente responsável pelas operações lógicas de interpretação e execução de intenções
  • UEG_PROVIDER: Provedor específico para a Universidade Estadual de Goiás
  • AUTH_SERVER: Servidor de autenticação e autorização externo

Metodologia de Desenvolvimento

A estrutua do projeto é distribuída entre projetos e módulos, promovendo flexibilidade e reutilização de componentes.

Plataformas Digitais Modernas

Evolução das Plataformas Digitais

A PIPA representa a evolução natural das plataformas digitais no contexto educacional, combinando a robustez dos sistemas tradicionais com a flexibilidade e inteligência dos assistentes virtuais modernos.

Vantagens Competitivas

  • Redução da curva de aprendizado para usuários
  • Acesso mais intuitivo aos serviços institucionais
  • Processamento inteligente de consultas em linguagem natural
  • Integração transparente com sistemas legados
  • Escalabilidade para diferentes instituições
